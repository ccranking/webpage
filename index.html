Quick Start

VSCode Extension

Install directly from VSCode:

  1. Open VSCode
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "ccusage"
  4. Click Install

Or install from command line:

code --install-extension ccusage.ccusage-ext

CLI Tool

Install via npm:

npm install -g ccusage

Basic usage:

ccusage analyze
ccusage report --format json
ccusage watch
