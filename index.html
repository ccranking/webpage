About CCRank

CCRank provides open-source analytics tools for Claude Code, helping developers track token usage and optimize costs in AI-assisted development. Our VSCode extension offers real-time monitoring and comprehensive reporting for all Claude models.

As a community-driven project, we rely on sponsorships to maintain and improve our tools. Your support enables us to develop new features and provide better analytics for the entire developer community.

Key Features

📊

Real-time Analytics

Track token usage and costs instantly with comprehensive reports for all Claude models.

🔧

VSCode Integration

Native extension with status bar monitoring and interactive dashboards right in your IDE.

💰

Cost Optimization

Accurate pricing calculations help you optimize usage and control AI development costs.

🌐

Open Source

MIT licensed with transparent development. Your sponsorship keeps it free for everyone.

Available Tools

ccusage-ext

VSCode Extension

A comprehensive VSCode extension that brings Claude Code usage tracking directly into your IDE. Features include real-time monitoring, cost analytics, and detailed reporting.

  • Live token counting in status bar
  • Interactive analytics dashboards
  • Automatic conversation detection
  • Multi-environment support
GitHub VSCode Marketplace

ccrank-web

Web Dashboard

A web-based dashboard for visualizing and analyzing Claude Code usage across teams and projects.

  • Team analytics
  • Project comparisons
  • Usage trends
  • Export capabilities
