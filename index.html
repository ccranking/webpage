About CCRank Project

Project Background

The CCRank project was born in the new era of AI-assisted development, with the goal of enabling developers to accurately understand their Claude Code usage and optimize costs.

We recognized that while Claude Code revolutionizes the development process, visualizing usage patterns and costs remained a significant challenge for developers.

Mission

To provide comprehensive analytics tools and insights that enable developers to effectively utilize AI tools and maximize their productivity.

Vision

To become the standard analytics platform for AI-assisted development, creating a future where developers worldwide can leverage AI more efficiently.

Available Tools

ccusage-ext

VSCode Extension

A comprehensive VSCode extension that brings Claude Code usage tracking directly into your IDE. Features include real-time monitoring, cost analytics, and detailed reporting.

  • Live token counting in status bar
  • Interactive analytics dashboards
  • Automatic conversation detection
  • Multi-environment support

ccrank-web

Web Dashboard

A web-based dashboard for visualizing and analyzing Claude Code usage across teams and projects.

Quick Start

VSCode Extension

Install directly from VSCode:

  1. Open VSCode
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "ccusage"
  4. Click Install
